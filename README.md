A cypress command to replace placeholders within json object or string

## Installation

To install with npm:

```
npm i cypress-placeholder-replacer-command --save
```

## Usage

Usage within Cypress:

```
import 'cypress-fill-command'

cy.replacePlaceholders ...

```
